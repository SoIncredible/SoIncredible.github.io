е…ідәҺејӮжӯҘзҡ„еҺҶеҸІ

C#ејҖеҸ‘иҖ…еңЁз¬¬дёҖж¬ЎжҺҘи§ҰејӮжӯҘзҡ„жҰӮеҝө, еә”иҜҘжҳҜйҖҡиҝҮTaskзұ»еһӢ. TaskжҳҜеңЁ.Net 4.0зҡ„ж—¶еҖҷиў«еј•е…Ҙзҡ„. дёҖдёӘtaskе°ұжҳҜдёҖдёӘworkеҚ•е…ғ, дё”иҜҘtaskжүҝиҜә, еңЁиҝҷдёӘtaskжңӘжқҘе®ҢжҲҗзҡ„ж—¶еҖҷ, дјҡе°Ҷз»“жһңиҝ”еӣһз»ҷtaskзҡ„и°ғз”ЁиҖ…. иҝҷдёӘTaskеҸҜиғҪжҳҜз”ұIOж“ҚдҪңж”ҜжҢҒжҲ–иҖ…и®Ўз®—еҜҶйӣҶеһӢж“ҚдҪң(иҝҷжӯЈеҘҪеҜ№еә”дәҶз¬”иҖ…еңЁеҚҸзЁӢйғЁеҲҶжүҖиҜҙзҡ„CPUеҜҶйӣҶеһӢж“ҚдҪңе’ҢIOеһӢж“ҚдҪң, UnityеҚҸзЁӢеҸӘиғҪз”ЁжқҘеҒҡIOеһӢзҡ„ејӮжӯҘж“ҚдҪң). йҮҚиҰҒзҡ„жҳҜиҜҘж“ҚдҪңзҡ„з»“жһңжҳҜиҮӘеҢ…еҗ«зҡ„пјҢдё”е…·жңүдёҖзӯүе…¬ж°‘иә«д»ҪгҖӮдҪ еҸҜд»ҘиҮӘз”ұдј йҖ’иҝҷдёӘ"жңӘжқҘ"пјҡе°Ҷе…¶еӯҳеӮЁеңЁеҸҳйҮҸдёӯгҖҒд»Һж–№жі•иҝ”еӣһе®ғгҖҒжҲ–дј йҖ’з»ҷе…¶д»–ж–№жі•гҖӮдҪ еҸҜд»Ҙе°ҶдёӨдёӘ"жңӘжқҘ"еҗҲе№¶еҪўжҲҗж–°зҡ„д»»еҠЎпјҢеҸҜд»ҘеҗҢжӯҘзӯүеҫ…з»“жһңпјҢд№ҹеҸҜд»ҘйҖҡиҝҮж·»еҠ "е»¶з»ӯ"жқҘ"зӯүеҫ…"з»“жһңгҖӮд»…еҮӯд»»еҠЎе®һдҫӢжң¬иә«пјҢдҪ е°ұиғҪеҶіе®ҡеңЁж“ҚдҪңжҲҗеҠҹгҖҒеӨұиҙҘжҲ–иў«еҸ–ж¶Ҳж—¶йҮҮеҸ–дҪ•з§ҚеӨ„зҗҶгҖӮ

д»»еҠЎе№¶иЎҢеә“ (TPL) ж”№еҸҳдәҶжҲ‘д»¬еҜ№е№¶еҸ‘зј–зЁӢзҡ„и®ӨзҹҘпјҢиҖҢ C# 5 йҖҡиҝҮеј•е…Ҙ async/await иҝӣдёҖжӯҘжҺЁеҠЁдәҶиҝҷдёҖеҸ‘еұ•гҖӮasync/await и®©д»»еҠЎзҡ„з»„еҗҲеҸҳеҫ—жӣҙеҠ е®№жҳ“пјҢе№¶е…Ғи®ёејҖеҸ‘иҖ…дҪҝз”ЁзҶҹжӮүзҡ„д»Јз Ғз»“жһ„пјҢеҰӮ try/catchгҖҒusing зӯүгҖӮдҪҶжҳҜasync/awaitд№ҹжңүе…¶ејҖй”Җ. иҰҒзҗҶи§Је…·дҪ“ејҖй”ҖжҳҜд»Җд№Ҳ, жҲ‘д»¬йңҖиҰҒж·ұе…Ҙеә•еұӮжңәеҲ¶дёӯ.

ејӮжӯҘж–№жі•еҶ…йғЁ

жҷ®йҖҡж–№жі•еҸӘжңүдёҖдёӘе…ҘеҸЈзӮ№е’ҢдёҖдёӘйҖҖеҮәзӮ№пјҲиҷҪз„¶еҸҜиғҪжңүеӨҡдёӘ return иҜӯеҸҘпјҢдҪҶеңЁиҝҗиЎҢж—¶пјҢжҜҸж¬Ўи°ғз”Ёд»…жңүдёҖдёӘе®һйҷ…йҖҖеҮәзӮ№пјүгҖӮдҪҶ ејӮжӯҘж–№жі•пјҲ*пјү е’Ң иҝӯд»ЈеҷЁж–№жі•пјҲеҢ…еҗ« yield return зҡ„ж–№жі•пјүеҲҷдёҚеҗҢгҖӮеҜ№дәҺејӮжӯҘж–№жі•иҖҢиЁҖпјҢи°ғз”Ёж–№еҮ д№ҺеҸҜд»Ҙз«ӢеҚіиҺ·еҸ–з»“жһңпјҲеҚі Task жҲ– TaskпјүпјҢз„¶еҗҺйҖҡиҝҮиҝ”еӣһзҡ„д»»еҠЎпјҲTaskпјүжқҘ вҖңawaitвҖқ ж–№жі•зҡ„е®һйҷ…жү§иЎҢз»“жһңгҖӮ

пјҲ*пјүжҲ‘д»¬е®ҡд№үзҡ„ вҖңејӮжӯҘж–№жі•вҖқ жҳҜжҢҮз”Ё async дёҠдёӢж–Үе…ій”®еӯ—ж Үи®°зҡ„ж–№жі•гҖӮиҝҷе№¶дёҚдёҖе®ҡж„Ҹе‘ізқҖиҜҘж–№жі•дјҡејӮжӯҘжү§иЎҢпјҢз”ҡиҮіе®Ңе…ЁдёҚж„Ҹе‘ізқҖе®ғжҳҜејӮжӯҘзҡ„гҖӮе®ғд»…иЎЁзӨә зј–иҜ‘еҷЁдјҡеҜ№иҜҘж–№жі•иҝӣиЎҢжҹҗдәӣзү№ж®ҠиҪ¬жҚўгҖӮ

зңӢдёҖдёӢдёӢйқўиҝҷдёӘејӮжӯҘж–№жі•:

дёҠйқўиҝҷж®өд»Јз Ғдёӯзҡ„дёӨдёӘж–№жі•йғҪдҪҝз”ЁдәҶasyncж Үи®°, еҲ°еӨҙжқҘйғҪеҸӘжҳҜдёәдәҶawait Task.Delay(42)иҝҷдёҖиЎҢд»Јз ҒжңҚеҠЎзҡ„.
дёәдәҶжӣҙеҘҪзҡ„зҗҶи§Јзј–иҜ‘еҷЁеҒҡдәҶжҲ–иҖ…иғҪеҒҡд»Җд№Ҳ, и®©жҲ‘д»¬е°қиҜ•зқҖжүӢеҶҷдёҖдёӢиҪ¬жҚўиҝҮзЁӢ

жүӢеҠЁи§Јжһ„ејӮжӯҘж–№жі•

TPLпјҲд»»еҠЎе№¶иЎҢеә“пјүжҸҗдҫӣдәҶдёӨеӨ§ж ёеҝғжһ„е»әжЁЎеқ—пјҢеё®еҠ©жҲ‘д»¬жһ„йҖ е’Ңз»„еҗҲд»»еҠЎпјҡ

  • д»»еҠЎе»¶з»ӯпјҲTask Continuationпјү-> йҖҡиҝҮ Task.ContinueWithе®һзҺ°
  • жүӢеҠЁжһ„е»әд»»еҠЎ -> йҖҡиҝҮ TaskCompletionSource<T> зұ»е®һзҺ°
    пјҲжіЁпјҡеүҚиҖ…з”ЁдәҺй“ҫејҸзј–жҺ’д»»еҠЎпјҢеҗҺиҖ…з”ЁдәҺжүӢеҠЁжҺ§еҲ¶д»»еҠЎз”ҹе‘Ҫе‘ЁжңҹгҖӮпјү

иҝҷж®өд»Јз ҒиҷҪз„¶еҶ—й•ҝпјҢдҪҶйҖ»иҫ‘зӣёеҜ№зӣҙжҺҘгҖӮеҺҹе…ҲеңЁ GetStockPriceForAsync дёӯзҡ„жүҖжңүйҖ»иҫ‘йғҪиў«иҪ¬з§»еҲ°дәҶ GetStockPriceForAsync_StateMachine.Start ж–№жі•дёӯпјҢиҜҘж–№жі•йҮҮз”ЁдәҶ"е»¶з»ӯдј йҖ’йЈҺж ј"(continuation passing style)гҖӮејӮжӯҘиҪ¬жҚўзҡ„ж ёеҝғз®—жі•е°ұжҳҜе°ҶеҺҹе§Ӣж–№жі•еңЁ await иҫ№з•ҢеӨ„еҲҶеүІжҲҗиӢҘе№Ід»Јз Ғеқ—пјҡ

  • з¬¬дёҖдёӘд»Јз Ғеқ—пјҡд»Һж–№жі•ејҖе§ӢеҲ°з¬¬дёҖдёӘ await д№Ӣй—ҙзҡ„д»Јз Ғ
  • з¬¬дәҢдёӘд»Јз Ғеқ—пјҡд»Һз¬¬дёҖдёӘ await еҲ°з¬¬дәҢдёӘ await д№Ӣй—ҙзҡ„д»Јз Ғ
  • з¬¬дёүдёӘд»Јз Ғеқ—пјҡд»Һз¬¬дәҢдёӘ await еҲ°з¬¬дёүдёӘ await жҲ–ж–№жі•з»“жқҹзҡ„д»Јз Ғ д»ҘжӯӨзұ»жҺЁ

жҜҸдёӘиў« await зҡ„д»»еҠЎзҺ°еңЁйғҪеҸҳжҲҗдәҶзҠ¶жҖҒжңәзҡ„дёҖдёӘеӯ—ж®өпјҢиҖҢ Start ж–№жі•дјҡе°ҶиҮӘе·ұжіЁеҶҢдёәиҝҷдәӣд»»еҠЎзҡ„е»¶з»ӯпјҲcontinuationпјүпјҡ

йҡҸеҗҺпјҢеҪ“д»»еҠЎе®ҢжҲҗж—¶пјҢStartж–№жі•дјҡиў«еӣһи°ғпјҢе№¶йҖҡиҝҮжЈҖжҹҘ_stateеӯ—ж®өжқҘзЎ®е®ҡеҪ“еүҚжү§иЎҢйҳ¶ж®өгҖӮжҺҘзқҖпјҢзҠ¶жҖҒжңәдјҡеҲӨж–ӯд»»еҠЎжҳҜжҲҗеҠҹе®ҢжҲҗгҖҒиў«еҸ–ж¶ҲиҝҳжҳҜеҮәзҺ°ејӮеёёгҖӮеҰӮжһңжҳҜжҲҗеҠҹе®ҢжҲҗзҡ„жғ…еҶөпјҢзҠ¶жҖҒжңәе°ұдјҡз»§з»ӯжү§иЎҢдёӢдёҖдёӘд»Јз Ғеқ—гҖӮеҪ“жүҖжңүж“ҚдҪңйғҪе®ҢжҲҗеҗҺпјҢзҠ¶жҖҒжңәдјҡи®ҫзҪ®TaskCompletionSourceе®һдҫӢзҡ„з»“жһңеҖјпјҢжӯӨж—¶д»ҺGetStockPricesForAsyncиҝ”еӣһзҡ„д»»еҠЎзҠ¶жҖҒе°ұдјҡеҸҳжӣҙдёәе·Іе®ҢжҲҗгҖӮ

иҝҷз§Қ"е®һзҺ°ж–№ејҸ"еӯҳеңЁеҮ дёӘдёҘйҮҚзјәйҷ·пјҡ

  • еӨ§йҮҸе ҶеҶ…еӯҳеҲҶй…Қпјҡ
    • зҠ¶жҖҒжңәе®һдҫӢйңҖиҰҒ1ж¬ЎеҶ…еӯҳеҲҶй…Қ
    • TaskCompletionSourceйңҖиҰҒ1ж¬ЎеҲҶй…Қ
    • TaskCompletionSourceеҶ…йғЁд»»еҠЎйңҖиҰҒ1ж¬ЎеҲҶй…Қ, жҢүз…§GPTзҡ„иҜҙжі•TaskCompletionSource<decimal>еҶ…йғЁдјҡиҮӘеҠЁеҲӣе»әдёҖдёӘTask<decimal> е®һдҫӢпјҲйҖҡиҝҮе…¶ .Task еұһжҖ§и®ҝй—®пјү
    • е»¶з»ӯе§”жүҳ(delegate)йңҖиҰҒ1ж¬ЎеҲҶй…Қ, жҢҮзҡ„еә”иҜҘжҳҜ _initializeMapIfNeededTask.ContinueWith(_ => Start());
  • зјәе°‘"зғӯи·Ҝеҫ„дјҳеҢ–"пјҡ
    еҪ“иў«зӯүеҫ…зҡ„д»»еҠЎе·Із»Ҹе®ҢжҲҗж—¶пјҢе®Ңе…ЁжІЎжңүеҝ…иҰҒеҲӣе»әе»¶з»ӯе§”жүҳ
  • еҸҜжү©еұ•жҖ§дёҚи¶іпјҡ
    е®һзҺ°дёҺTaskзұ»зҙ§еҜҶиҖҰеҗҲпјҢеҜјиҮҙж— жі•з”ЁдәҺе…¶д»–еңәжҷҜпјҢдҫӢеҰӮпјҡ
    • зӯүеҫ…йқһTaskзұ»еһӢ
    • иҝ”еӣһйқһTask/Taskзұ»еһӢ

дёӢйқўжҲ‘д»¬е°ұжқҘзңӢдёҖдёӢдёҠиҝ°жҸҗеҲ°зҡ„зјәйҷ·еңЁзңҹжӯЈзҡ„ејӮжӯҘжңәеҲ¶дёӯжҳҜеҰӮдҪ•иў«и§ЈеҶізҡ„.

ејӮжӯҘжңәеҲ¶

зј–иҜ‘еҷЁжү§иЎҢејӮжӯҘж–№жі•иҪ¬жҚўзҡ„ж•ҙдёӘиҝҮзЁӢ, е…¶е®һе·Із»Ҹе’ҢдёҠйқўжҲ‘д»¬жүӢеҠЁе®һзҺ°зҡ„ж–№ејҸеҚҒеҲҶжҺҘиҝ‘дәҶ. дёәдәҶеҫ—еҲ°йў„жңҹзҡ„иЎҢдёә, зј–иҜ‘еҷЁдҫқиө–дәҺд»ҘдёӢзҡ„зұ»еһӢ:

  • еҜ№дәҺдёҖдёӘејӮжӯҘж–№жі•зҡ„з”ҹжҲҗзҡ„зҠ¶жҖҒжңә, е…¶иЎҢдёәдјҡеғҸж Ҳеё§дёҖж ·, е№¶дё”иҜҘзҠ¶жҖҒжңәеҢ…еҗ«еҺҹе§ӢејӮжӯҘж–№жі•дёӯзҡ„е…ЁйғЁйҖ»иҫ‘
  • AsyncTaskMethodBuilder<T>жҢҒжңүе®ҢжҲҗзҡ„taskпјҲдёҺ TaskCompletionSource зұ»еһӢйқһеёёзӣёдјјпјү, е№¶дё”з®ЎзҗҶзҠ¶жҖҒжңәзҡ„иҪ¬жҚў.
  • TaskAwaiter<T>иҙҹиҙЈеҢ…иЈ…task, е№¶еңЁйңҖиҰҒж—¶и°ғеәҰе…¶е»¶з»ӯ
  • MoveNetRunnerиҙҹиҙЈеңЁжӯЈзЎ®зҡ„жү§иЎҢдёҠдёӢж–Үдёӯи°ғз”ЁIAsyncStateMachine.MoveNextж–№жі•

з”ҹжҲҗзҡ„зҠ¶жҖҒжңәеңЁDebugжЁЎејҸдёӢжҳҜдёҖдёӘзұ», еңЁReleaseжЁЎејҸдёӢеҲҷжҳҜдёҖдёӘstruct. е…¶д»–зҡ„зұ»еһӢ(йҷӨдәҶMoveNextRunnerзұ»)еңЁBCLдёӯеқҮиў«е®ҡд№үдёәдәҶstruct.

зј–иҜ‘еҷЁз”ҹжҲҗзҡ„зҠ¶жҖҒжңәзұ»еһӢеҗҚз§°йҖҡеёёзұ»дјјдәҺ<YourMethodNameAsync>d_1, дёәдәҶйҒҝе…ҚеҗҚз§°еҶІзӘҒ, з”ҹжҲҗзҡ„еҗҚеӯ—дёӯеҢ…еҗ«дәҶдёҖдәӣж— ж•Ҳзҡ„ж ҮиҜҶз¬Ұеӯ—з¬Ұ, иҝҷдәӣеӯ—з¬Ұж— жі•иў«з”ЁжҲ·зӣҙжҺҘе®ҡд№үжҲ–еј•з”Ё. дҪҶжҳҜдёәдәҶз®ҖеҢ–иҜҙжҳҺ, еңЁеҗҺз»ӯзҡ„жүҖжңүзӨәдҫӢдёӯ, жҲ‘д»¬иҝҳжҳҜе°ҶдҪҝз”Ёжңүж•Ҳзҡ„ж ҮиҜҶз¬ҰжқҘжӣҝжҚўжҺү<гҖҒ>гҖҒ_иҝҷдәӣеӯ—з¬Ұ, е№¶дё”дҪҝз”Ёжӣҙе®№жҳ“зҗҶи§Јзҡ„еҗҚеӯ—.

еҺҹе§Ӣж–№жі•

еҺҹе§Ӣзҡ„ејӮжӯҘж–№жі•дјҡеҲӣе»әдёҖдёӘзҠ¶жҖҒжңәе®һдҫӢ, з”ЁжҚ•иҺ·зҡ„зҠ¶жҖҒпјҲеҰӮжһңж–№жі•йқһйқҷжҖҒпјҢеҲҷеҢ…жӢ¬ this жҢҮй’ҲпјүеҲқе§ӢеҢ–иҜҘе®һдҫӢ, йҖҡиҝҮдј йҖ’зҠ¶жҖҒжңәе®һдҫӢзҡ„еј•з”Ёе№¶и°ғз”ЁAsyncTaskMethodBuilder<T>.StartжқҘеҗҜеҠЁжү§иЎҢ

дј йҖ’еј•з”ЁжҳҜдёҖдёӘйҮҚиҰҒзҡ„дјҳеҢ–, еӣ дёәдёҖдёӘзҠ¶жҖҒжңәеҫҖеҫҖжҳҜзӣёеҜ№еӨ§зҡ„з»“жһ„(еӨ§дәҺ100bytes), еӣ жӯӨдј йҖ’еј•з”ЁдјҡйҒҝе…ҚеҶ—дҪҷжӢ·иҙқ.
зҠ¶жҖҒжңәд»Јз ҒеҰӮдёӢ:

з”ҹжҲҗзҡ„зҠ¶жҖҒжңәд»Јз ҒзңӢиө·жқҘжҜ”иҫғеӨҚжқӮ, дҪҶжҳҜжң¬иҙЁдёҠ, е®ғе’ҢжҲ‘д»¬жүӢеҶҷзҡ„йӮЈдёҖзүҲзҡ„д»Јз ҒжҳҜеҚҒеҲҶзӣёдјјзҡ„.
еҚідҪҝдёӨдёӘзүҲжң¬еҚҒеҲҶзӣёдјј, дҪҶжҳҜиҝҳжҳҜжңүдёҖдәӣйҮҚиҰҒзҡ„дёҚеҗҢзҡ„.

  1. вҖңHot PathвҖқ дјҳеҢ–

дёҚеғҸжҲ‘д»¬еҺҹз”ҹзҡ„ж–№ејҸ, з”ҹжҲҗзҡ„зҠ¶жҖҒжңәзҹҘйҒ“дёҖдёӘиў«awaitedзҡ„taskеҸҜиғҪе·Із»Ҹе®ҢжҲҗдәҶ

еҰӮжһңиҝҷдёӘиў«awaitзҡ„taskе·Із»Ҹе®ҢжҲҗдәҶ(ж— и®әжҲҗеҠҹдёҺеҗҰ), зҠ¶жҖҒжңәдјҡеҗ‘еүҚзӣҙиЎҢдёӢдёҖжӯҘ

иҝҷе°ұж„Ҹе‘ізқҖ, еҰӮжһңжүҖжңүawaitedзҡ„taskйғҪе·Із»Ҹе®ҢжҲҗдәҶ, ж•ҙдёӘзҠ¶жҖҒжңәе°ҶеҸӘдјҡеҒңз•ҷеңЁж ҲдёҠ. дёҖдёӘејӮжӯҘж–№жі•з”ҡиҮіеңЁд»ҠеӨ©еҰӮжһңжүҖжңүawaitedзҡ„taskйғҪе·Із»Ҹе®ҢжҲҗжҲ–иҖ…еҗҢжӯҘе®ҢжҲҗдәҶ, йӮЈд№ҲиҝҷдёӘејӮжӯҘж–№жі•жҳҜеҸҜд»ҘеҸӘдјҡйҖ жҲҗеҫҲе°Ҹзҡ„еҶ…еӯҳејҖй”Җзҡ„. д»…еү©зҡ„ејҖй”ҖеҸӘжҳҜиҝҷдёӘtaskжң¬иә«.

  1. й”ҷиҜҜеӨ„зҗҶ

еҪ“еүҚйҖ»иҫ‘е№¶жңӘдё“й—ЁеӨ„зҗҶд»»еҠЎеӨ„дәҺж•…йҡңзҠ¶жҖҒжҲ–е·ІеҸ–ж¶ҲзҠ¶жҖҒзҡ„жғ…еҶөгҖӮзҠ¶жҖҒжңәйҖҡиҝҮи°ғз”Ёawaiter.GetResult()ж–№жі•пјҢеҪ“д»»еҠЎиў«еҸ–ж¶Ҳж—¶е°ҶжҠӣеҮәTaskCancelledExceptionпјҢиӢҘд»»еҠЎеӨұиҙҘеҲҷжҠӣеҮәе…¶д»–ејӮеёёгҖӮиҝҷз§Қи®ҫи®ЎеҚҒеҲҶдјҳйӣ…пјҢеӣ дёәGetResult()еңЁй”ҷиҜҜеӨ„зҗҶжңәеҲ¶дёҠдёҺtask.Wait()жҲ–task.Resultжңүжң¬иҙЁеҢәеҲ«гҖӮ

ж— и®әжҳҜtask.Wait()иҝҳжҳҜtask.ResultпјҢеҚідҫҝд»»еҠЎеӣ еҚ•дёҖејӮеёёеҜјиҮҙеӨұиҙҘпјҢе®ғд»¬йғҪдјҡжҠӣеҮәAggregateExceptionгҖӮиҝҷиғҢеҗҺзҡ„йҖ»иҫ‘еҫҲз®ҖеҚ•пјҡд»»еҠЎдёҚд»…еҸҜиғҪд»ЈиЎЁйҖҡеёёеҸӘжңүеҚ•дёҖж•…йҡңзҡ„IOж“ҚдҪңпјҢд№ҹеҸҜиғҪжҳҜе№¶иЎҢи®Ўз®—зҡ„з»“жһңгҖӮеҗҺиҖ…еҸҜиғҪдә§з”ҹеӨҡдёӘй”ҷиҜҜпјҢиҖҢAggregateExceptionжӯЈжҳҜдёәиҒҡеҗҲжүҖжңүй”ҷиҜҜиҖҢи®ҫи®ЎгҖӮ

дҪҶasync/awaitжЁЎејҸдё“дёәејӮжӯҘж“ҚдҪңи®ҫи®ЎпјҢиҝҷзұ»ж“ҚдҪңйҖҡеёёжңҖеӨҡеҸӘдјҡдә§з”ҹдёҖдёӘй”ҷиҜҜгҖӮеӣ жӯӨиҜӯиЁҖи®ҫи®ЎиҖ…и®ӨдёәпјҢиӢҘawaiter.GetResult()иғҪеҜ№AggregateExceptionиҝӣиЎҢи§ЈеҢ…е№¶д»…жҠӣеҮәйҰ–дёӘејӮеёёпјҢе°Ҷжӣҙз¬ҰеҗҲдҪҝз”ЁеңәжҷҜгҖӮиҝҷдёҖи®ҫи®Ўе№¶йқһе®ҢзҫҺпјҢжҲ‘д»¬е°ҶеңЁеҗҺз»ӯж–Үз« дёӯзңӢеҲ°иҝҷз§ҚжҠҪиұЎж–№жЎҲеҸҜиғҪеӯҳеңЁзҡ„жјҸжҙһгҖӮ

ејӮжӯҘзҠ¶жҖҒжңәеҸӘжҳҜж•ҙдёӘжӢјеӣҫзҡ„дёҖйғЁеҲҶгҖӮиҰҒе®Ңж•ҙзҗҶи§Је…¶иҝҗдҪңжңәеҲ¶пјҢжҲ‘д»¬иҝҳйңҖдәҶи§ЈзҠ¶жҖҒжңәе®һдҫӢеҰӮдҪ•дёҺTaskAwaiter<T>е’ҢAsyncTaskMethodBuilder<T>иҝӣиЎҢдәӨдә’гҖӮ

иҝҷдәӣжЁЎеқ—жҳҜеҰӮдҪ•иў«иҒ”зі»еҲ°дёҖиө·зҡ„е‘ў?

еӣҫиЎЁзңӢдјјиҝҮдәҺеӨҚжқӮпјҢдҪҶжҜҸдёӘз»„д»¶йғҪз»ҸиҝҮзІҫеҝғи®ҫи®Ўдё”жү®жј”зқҖйҮҚиҰҒи§’иүІгҖӮжңҖжңүи¶Јзҡ„еҚҸдҪңеҸ‘з”ҹеңЁзӯүеҫ…зҡ„д»»еҠЎе°ҡжңӘе®ҢжҲҗж—¶пјҲеӣҫдёӯз”ЁжЈ•иүІзҹ©еҪўж Үи®°пјүпјҡ

зҠ¶жҖҒжңәжіЁеҶҢд»»еҠЎе»¶з»ӯ

  • зҠ¶жҖҒжңәи°ғз”Ё __builder.AwaitUnsafeOnCompleted(ref awaiter, ref this)пјҢе°ҶиҮӘиә«жіЁеҶҢдёәд»»еҠЎзҡ„е»¶з»ӯгҖӮ
  • AsyncTaskMethodBuilder зЎ®дҝқд»»еҠЎе®ҢжҲҗеҗҺи°ғз”Ё IAsyncStateMachine.MoveNext ж–№жі•(иҜ‘жіЁ:иҝҷеҸҘиҜқзҡ„ж„ҸжҖқжҳҜиҜҙ, еҸӘиҰҒдҪ , жӣҙеҮҶзЎ®ең°иҜҙиҝҷж®өд»Јз ҒжҳҜзј–иҜ‘еҷЁз”ҹжҲҗзҡ„, и°ғз”ЁдәҶAwaitUnsafeOnCompletedжҺҘеҸЈ, йӮЈд№Ҳзј–иҜ‘еҷЁе°ұиғҪдҝқиҜҒд»»еҠЎе®ҢжҲҗеҗҺдјҡи°ғз”ЁIAsyncStateMachine.MoveNextж–№жі•дәҶ)пјҡ
    • жҚ•иҺ·еҪ“еүҚ жү§иЎҢдёҠдёӢж–ҮпјҲExecutionContextпјүпјҢеҲӣе»әдёҖдёӘ MoveNextRunner е®һдҫӢпјҢе°Ҷе…¶дёҺеҪ“еүҚзҠ¶жҖҒжңәе®һдҫӢе…іиҒ”гҖӮ
    • д»ҺMoveNextRunner.RunеҲӣе»әдёҖдёӘ Action е§”жүҳпјҢз”ЁдәҺеңЁжҚ•иҺ·зҡ„жү§иЎҢдёҠдёӢж–ҮдёӯжҺЁиҝӣзҠ¶жҖҒжңәгҖӮи°ғз”Ё TaskAwaiter.UnsafeOnCompleted(action)пјҢе°ҶдёҠиҝ° Action е§”жүҳи°ғеәҰдёәзӯүеҫ…д»»еҠЎзҡ„е»¶з»ӯ(иҜ‘жіЁ: еҸҜд»ҘеҺ»зңӢдёҖдёӢC#жәҗз Ғдёӯзҡ„MoveNextRunnerе’ҢAsyncMethodBuilderCoreзҡ„е®һзҺ°, е…¶дёӯжё…жҷ°ең°еұ•зӨәдәҶеҰӮдҪ•жҚ•иҺ·еҪ“еүҚжү§иЎҢдёҠдёӢж–ҮгҖҒе°ҶIAsyncStateMachine.MoveNextж–№жі•е’Ңжү§иЎҢдёҠдёӢж–ҮдёҖиө·еҢ…иЈ…иҝӣMoveNextRunner.Runз”ҹжҲҗзҡ„Actionдёӯ)гҖӮ
      еҪ“зӯүеҫ…зҡ„д»»еҠЎе®ҢжҲҗж—¶пјҢжіЁеҶҢзҡ„еӣһи°ғпјҲAction е§”жүҳпјүиў«и§ҰеҸ‘пјҢзҠ¶жҖҒжңәз»§з»ӯжү§иЎҢејӮжӯҘж–№жі•зҡ„дёӢдёҖж®өд»Јз Ғеқ—гҖӮ

иҝҷеј ж—¶еәҸеӣҫдёӯAsyncTaskMethodBuilderзҡ„Startж–№жі•зҡ„е…·дҪ“е®һзҺ°еңЁдёҠйқўзј–иҜ‘еҷЁз”ҹжҲҗзҡ„д»Јз ҒдёӯдҪ“зҺ°, з¬”иҖ…жҠҠиҝҷйғЁеҲҶд»Јз ҒиҙҙеҲ°дёӢйқў:

е…¶е®һе°ұжҳҜи°ғз”ЁдәҶзҠ¶жҖҒжңәзҡ„MoveNextж–№жі•, з„¶еҗҺеӣһеҲ°зј–иҜ‘еҷЁдёәжҲ‘д»¬з”ҹжҲҗзҡ„д»Јз Ғдёӯ, еңЁзҠ¶жҖҒжңәзҡ„MoveNextж–№жі•йҮҢ, и°ғз”ЁдәҶ __builder.AwaitUnsafeOnCompleted(ref awaiter, ref this);жҲ‘д»¬зңӢдёҖдёӢAsyncTaskMethodBuilderдёӯзҡ„AwaitUnsafeOnCompletedзҡ„е…·дҪ“е®һзҺ°:

иҝҷдёӢе°ұе’Ңж—¶еәҸеӣҫдёӯзҡ„еҶ…е®№еҜ№дёҠдәҶ:
var continuation = m_coreState.GetCompletionAction(AsyncCausalityTracer.LoggingOn ? this.Task : null, ref runnerToInitialize); еҜ№еә”дәҶж—¶еәҸеӣҫдёӯMoveNextRunnerйғЁеҲҶ.

awaiter.UnsafeOnCompleted(continuation);еҜ№еә”дәҶж—¶еәҸеӣҫдёӯзҡ„TaskAwaiterйғЁеҲҶ, еҪ“иҝҷдёӘawaiter completeдәҶд№ӢеҗҺ, е°ұдјҡи§ҰеҸ‘дј е…Ҙзҡ„continuation, д№ҹе°ұжҳҜstateMachineзҡ„MoveNext. ж•ҙдёӘжөҒзЁӢе°ұи·‘йҖҡдәҶ.

еңЁжҲ‘д»¬зӨәдҫӢдёӯиҝҷдёӘAwaiterжҳҜдёҖдёӘTaskAwaiter, е…¶UnsafeOnCompletedжҺҘеҸЈзҡ„е®һзҺ°еҰӮдёӢ:

C#жәҗз ҒзңӢеҲ°иҝҷйҮҢе…¶е®һе°ұе·®дёҚеӨҡдәҶ, еҶҚеҫҖдёӢзңӢи°ғз”ЁеҜ№дәҺC#зҡ„дҪҝз”ЁиҖ…ж„Ҹд№үе°ұдёҚжҳҜеҫҲеӨ§дәҶ.

жү§иЎҢдёҠдёӢж–Ү

й—®йўҳпјҡд»Җд№ҲжҳҜжү§иЎҢдёҠдёӢж–ҮпјҹдёәдҪ•йңҖиҰҒиҝҷз§ҚеӨҚжқӮзҡ„и®ҫи®Ўпјҹ
еңЁеҗҢжӯҘд»Јз ҒдёӯпјҢжҜҸдёӘзәҝзЁӢйҖҡиҝҮ зәҝзЁӢжң¬ең°еӯҳеӮЁпјҲThread-Local Storageпјү з»ҙжҠӨзҺҜеўғдҝЎжҒҜпјҢдҫӢеҰӮпјҡе®үе…ЁеҮӯжҚ®пјҲеҰӮ SecurityContextпјүеҢәеҹҹжҖ§и®ҫзҪ®пјҲеҰӮ CultureInfoпјүжҲ–иҖ…е…¶д»–дёҠдёӢж–Үж•°жҚ®пјҲеҰӮ AsyncLocal зҡ„еҖјпјү
еҪ“дёүдёӘж–№жі•еңЁеҗҢдёҖдёӘзәҝзЁӢдёӯдҫқж¬Ўи°ғз”Ёж—¶пјҢиҝҷдәӣдҝЎжҒҜдјҡиҮӘеҠЁеңЁж–№жі•й—ҙжөҒеҠЁгҖӮдҪҶжҳҜеҜ№дәҺејӮжӯҘж–№жі•жқҘиҜҙе°ұдёҚеҶҚжҳҜиҝҷж ·дәҶ. ејӮжӯҘж–№жі•зҡ„жҜҸдёӘд»Јз Ғж®өпјҲеҰӮ await еүҚеҗҺзҡ„д»Јз ҒпјүеҸҜиғҪеңЁдёҚеҗҢзәҝзЁӢдёҠжү§иЎҢпјҢзәҝзЁӢжң¬ең°еӯҳеӮЁ(Thread-Local Storage)еӨұж•ҲгҖӮ
еӣ жӯӨжү§иЎҢдёҠдёӢж–ҮжҙҫдёҠз”Ёеңә, е®ғдёә дёҖдёӘйҖ»иҫ‘жҺ§еҲ¶жөҒ з»ҙжҠӨдёҠдёӢж–ҮдҝЎжҒҜпјҢеҚідҪҝиҜҘжҺ§еҲ¶жөҒи·Ёи¶ҠеӨҡзәҝзЁӢгҖӮ
дҫӢеҰӮпјҢTask.Run жҲ– ThreadPool.QueueUserWorkItem дјҡ иҮӘеҠЁжҚ•иҺ·и°ғз”ЁзәҝзЁӢзҡ„жү§иЎҢдёҠдёӢж–ҮпјҢе№¶е°Ҷе…¶дёҺд»»еҠЎз»‘е®ҡгҖӮ
еҪ“д»»еҠЎжү§иЎҢж—¶пјҢи°ғеәҰеҷЁпјҲеҰӮ TaskSchedulerпјүйҖҡиҝҮ ExecutionContext.Run еңЁжҚ•иҺ·зҡ„дёҠдёӢж–ҮдёӯиҝҗиЎҢе§”жүҳпјҢзЎ®дҝқзҺҜеўғдҝЎжҒҜпјҲеҰӮе®үе…ЁеҮӯжҚ®пјүж— зјқе»¶з»ӯгҖӮ

йҖҡиҝҮдёӢйқўиҝҷдёӘдҫӢеӯҗзҗҶи§ЈдёҖдёӢ:

еңЁиҝҷдёӘдҫӢеӯҗдёӯ, жү§иЎҢдёҠдёӢж–ҮйҖҡиҝҮTask.RunжөҒеҗ‘дәҶTask.ContinueWith. еӣ жӯӨдёҠйқўиҝҷж®өд»Јз Ғзҡ„жү§иЎҢз»“жһңеҰӮдёӢ:

дҪҶжҳҜдёҚжҳҜжүҖжңүеңЁBCLдёӯзҡ„ж–№жі•йғҪдјҡиҮӘеҠЁжҚ•иҺ·е№¶жҒўеӨҚжү§иЎҢдёҠдёӢж–Ү. дёӨдёӘдҫӢеӨ–жҳҜ: TaskAwaiter<T>.UnsafeOnCompledteе’ҢAsyncMethodBuilder<T>.AwaitUnsafeOnComplete. иҝҷзңӢиө·жқҘеҚҒеҲҶеҘҮжҖӘ, иҜӯиЁҖи®ҫи®ЎиҖ…еҶіе®ҡж·»еҠ дёҚе®үе…Ёзҡ„ж–№жі•жүӢеҠЁдҪҝз”ЁAsyncMethodBuilder<T>е’ҢMoveNetRunnerиҖҢдёҚжҳҜдҫқиө–дәҺеҶ…е»әзҡ„зұ»дјјAwaitTaskContinuationиҝҷж ·зҡ„жңәеҲ¶й©ұеҠЁжү§иЎҢдёҠдёӢж–Ү, зҢңжөӢиҝҷжҳҜеҮәдәҺжҖ§иғҪй—®йўҳиҖғиҷ‘жҲ–иҖ…жҳҜеҜ№зҺ°жңүе®һзҺ°зҡ„еҸҰдёҖдёӘеҰҘеҚҸ.

иҫ“еҮәз»“жһңжҳҜ:

з»“и®ә

  • ејӮжӯҘж–№жі•пјҲasyncпјүзҡ„еә•еұӮиЎҢдёәдёҺеҗҢжӯҘж–№жі•жҲӘз„¶дёҚеҗҢпјҢе…¶ж ёеҝғжңәеҲ¶дҫқиө–дәҺзј–иҜ‘еҷЁз”ҹжҲҗзҡ„ зҠ¶жҖҒжңәпјҲState Machineпјүпјҡ

  • зј–иҜ‘еҷЁдјҡдёәжҜҸдёӘејӮжӯҘж–№жі•з”ҹжҲҗдёҖдёӘзӢ¬з«Ӣзҡ„зҠ¶жҖҒжңәпјҢе°ҶеҺҹж–№жі•зҡ„е…ЁйғЁйҖ»иҫ‘иҪ¬з§»иҮізҠ¶жҖҒжңәдёӯгҖӮ зҠ¶жҖҒжңәиҙҹиҙЈи·ҹиёӘжү§иЎҢиҝӣеәҰпјҲйҖҡиҝҮзҠ¶жҖҒеҖјпјүгҖҒжҢӮиө·пјҲawait ж—¶пјүдёҺжҒўеӨҚпјҲд»»еҠЎе®ҢжҲҗж—¶пјүзҡ„йҖ»иҫ‘жөҒиҪ¬гҖӮ

  • еҜ№еҗҢжӯҘе®ҢжҲҗзҡ„ж·ұеәҰдјҳеҢ–. иӢҘжүҖжңүзӯүеҫ…зҡ„д»»еҠЎпјҲawait зҡ„д»»еҠЎпјүе·ІеҗҢжӯҘе®ҢжҲҗпјҲеҰӮзј“еӯҳе‘ҪдёӯгҖҒеҶ…еӯҳи®Ўз®—зӯүж— йҳ»еЎһж“ҚдҪңпјүпјҢејӮжӯҘж–№жі•зҡ„жҖ§иғҪејҖй”ҖжһҒдҪҺпјҢеҮ д№ҺдёҺеҗҢжӯҘж–№жі•ж— ејӮгҖӮ
