RectTransformз»§жүҝиҮӘTransform, е№¶жІЎжңүеҢ…еҗ«еңЁUGUIдҪ“зі»йҮҢйқў, дҪҶжҳҜRectTransformзҡ„еӯ—ж®өз»қеӨ§еӨҡж•°жғ…еҶөдёӢеҸӘеңЁUGUIдҪ“зі»дёӯжүҚжңүж„Ҹд№ү. еӣ жӯӨжҲ‘д»¬еҸҜд»Ҙе°ҶRectTransformзңӢдҪңжҳҜUnity UGUIзҡ„дёҖйғЁеҲҶ, е№¶дё”е°Ҷе…¶дҪңдёәUGUIзі»еҲ—дёӯйҰ–дёӘеҲҶжһҗзҡ„еҜ№иұЎ.

й”ҡзӮ№(Anchor)дёҺдёӯеҝғзӮ№(Pivot)

й”ҡзӮ№Anchor

иҝҷдёӘAnchorе№¶дёҚжҳҜзӣҙжҺҘеҸӮдёҺдҪҚзҪ®дҝЎжҒҜиҝҗз®—зҡ„ж•°жҚ®, Unityдјҡе…Ҳж №жҚ®иҝҷдёӘдҝЎжҒҜи®Ўз®—еҮәдёҖдёӘanchorReferencePosition, еҶҚз”ЁиҝҷдёӘеҖјиҝӣиЎҢе…¶д»–зҡ„иҝҗз®—.
еңЁUnityдёӯ й”ҡзӮ№жҳҜеҸҜд»ҘиҮіеӨҡиў«еҲҶжҲҗеӣӣдёӘи§’зҡ„, е№¶дёҚжҳҜдёҖдёӘзӮ№. иҖҢеңЁи®Ўз®—дёҖдёӘRectTransformдёӯзҡ„дёҖдәӣеұһжҖ§ж—¶йңҖиҰҒжҠҠй”ҡзӮ№дҪңдёәдёҖдёӘ"зӮ№" еҸӮдёҺиҝҗз®—, дҪңдёәеӣӣдёӘи§’дёҚйҮҚеҗҲзҡ„жғ…еҶө, йңҖиҰҒжңүдёҖдёӘAnchor Reference PositionжқҘдҪңдёәй”ҡзӮ№,
Anchor Reference Positionзҡ„и®Ўз®—е…¬ејҸеҰӮдёӢ:
$$ AnchorReferencePosition_x = (1 - pivot_x) \times x_0 + pivot_x \times x_1 $$
$$ AnchorReferencePosition_y = (1 - pivot_y) \times y_0 + pivot_y \times y_1 $$
е…¶дёӯ (x0, y0)д»ЈиЎЁзҡ„жҳҜAnchorMinеңЁиҜҘRectTransformзҡ„зҲ¶иҠӮзӮ№дёӯзҡ„дҪҚзҪ®, (x1, y1)д»ЈиЎЁзҡ„жҳҜAnchorMaxеңЁиҜҘRectTransformзҡ„зҲ¶иҠӮзӮ№дёӯзҡ„дҪҚзҪ®, (pivotx, pivoty)жҢҮзҡ„жҳҜдёӯеҝғзӮ№зҡ„еҪ’дёҖеҢ–дҪҚзҪ®еқҗж Ү

жңүдёҖдёӘжҰӮеҝөеҸ«AnchoredPosition, иҝҷдёӘеҖјзҡ„еҗ«д№үжҳҜPivotзӮ№зӣёеҜ№дәҺAnchor Reference Positionзҡ„и·қзҰ», еҪ“й”ҡзӮ№зҡ„еӣӣдёӘи§’жІЎжңүйҮҚеҗҲеңЁдёҖзӮ№ж—¶, жҲ‘д»¬еҸҜиғҪдјҡд»ҘдёәAnchor Reference Position

AnchoredPositionе’ҢLocal Positionзҡ„еҢәеҲ«жҳҜд»Җд№Ҳ?
еңЁTransform(йқһRectTransform)дҪ“зі»дёӯ, еӯҗзү©дҪ“зҡ„localPositionеҸӘиғҪйҖҡиҝҮи°ғж•ҙе…¶е’ҢзҲ¶зү©дҪ“зҡ„зӣёеҜ№дҪҚзҪ®жқҘе®һзҺ°еҸҳеҢ–, еңЁRectTransformдҪ“зі»дёӢ, еӯҗзү©дҪ“зҡ„localPositionйҷӨдәҶеҸ—зӣёеҜ№дҪҚзҪ®зҡ„еҪұе“Қ, иҝҳдёҺзҲ¶зү©дҪ“зҡ„widthгҖҒеӯҗзү©дҪ“жң¬иә«зҡ„pivotжңүе…і

RectTransformдёӯзҡ„еӯ—ж®ө

рҹ’Ўжң¬ж–ҮжҲ‘д»¬еҸӘи®Ёи®әRectTransformдёӯжңүиҖҢTransformдёӯжІЎжңүзҡ„еӯ—ж®ө

  • rect Rect жіЁж„ҸRectTransformдёӯзҡ„rectжҲҗе‘ҳдёҚж”ҜжҢҒзӣҙжҺҘдҝ®ж”№жіЁж„Ҹд»ҘдёҠеӯ—ж®өйғҪжҳҜдёҚеҸҜд»ҘйҖҡиҝҮRectTransformж“ҚдҪңе…¶еҖјзҡ„., еӣ дёәжІЎжңүsetеұһжҖ§, еӨ–йғЁжІЎжңүеҜ№е®ғзҡ„и®ҝй—®жқғйҷҗ
    • xMin, д»ЈиЎЁзҡ„жҳҜrectеӣҙжҲҗзҹ©еҪўеҢәеҹҹзҡ„е·ҰдёӢи§’зҡ„жЁӘеқҗж ҮеҖј, еқҗж Үзі»еҺҹзӮ№жҳҜиҝҷдёӘrectеӣҙжҲҗзҡ„зҹ©еҪўеҢәеҹҹзҡ„дёӯеҝғзӮ№,
    • yMin, д»ЈиЎЁзҡ„жҳҜrectеӣҙжҲҗзҹ©еҪўеҢәеҹҹзҡ„е·ҰдёӢи§’зҡ„зәөеқҗж ҮеҖј, еқҗж Үзі»еҺҹзӮ№жҳҜиҝҷдёӘrectеӣҙжҲҗзҡ„зҹ©еҪўеҢәеҹҹзҡ„дёӯеҝғзӮ№,
    • mHeight, иҝҷдёӘrectеӣҙжҲҗзҹ©еҪўзҡ„й«ҳ
    • mWidth, иҝҷдёӘrectеӣҙжҲҗзҹ©еҪўзҡ„е®Ҫ
  • anchorMin Vector2 д»ҘеҪ“еүҚRectTransformзҡ„зҲ¶иҠӮзӮ№зҡ„rectзҡ„е·ҰдёӢи§’дёәеқҗж ҮеҺҹзӮ№, й”ҡзӮ№е·ҰдёӢи§’зҡ„еҪ’дёҖеҢ–дҪҚзҪ®еқҗж Ү
  • anchorMax Vector2 д»ҘеҪ“еүҚRectTransformзҡ„зҲ¶иҠӮзӮ№зҡ„rectзҡ„е·ҰдёӢи§’дёәеқҗж ҮеҺҹзӮ№, й”ҡзӮ№еҸідёҠи§’зҡ„еҪ’дёҖеҢ–дҪҚзҪ®еқҗж Ү
  • sizeDelta Vector2 еҪ“й”ҡзӮ№еӣӣи§’йҮҚеҗҲж—¶, sizeDelta.x=rect.width=(offsetMax-offsetMin).x, sizeDelta.y=rect.height=(offsetMax-offsetMin).y еҪ“дёҚйҮҚеҗҲж—¶ sizeDelta.x=(offsetMax-offsetMin).x, sizeDelta.y=(offsetMax-offsetMin).y
  • offsetMin Vector2 жҢҮй”ҡзӮ№е·ҰдёӢи§’дёҺrectзҡ„е·ҰдёӢи§’зҡ„еҒҸз§»еҖј д»ҘеҪ“еүҚRectTransformзҡ„зҲ¶иҠӮзӮ№зҡ„rectзҡ„е·ҰдёӢи§’дёәеқҗж ҮеҺҹзӮ№
  • offsetMax Vector2 жҢҮй”ҡзӮ№еҸідёҠи§’дёҺrectзҡ„еҸідёҠи§’зҡ„еҒҸз§»еҖј д»ҘеҪ“еүҚRectTransformзҡ„зҲ¶иҠӮзӮ№зҡ„rectзҡ„е·ҰдёӢи§’дёәеқҗж ҮеҺҹзӮ№
  • anchoredPosition Vector2
  • anchoredPosition3D Vector3 еҢ…еҗ«anchoredPositionзҡ„xyеҲҶйҮҸ, еўһеҠ дәҶzеҲҶйҮҸ дёҖиҲ¬з”ЁдёҚеҲ°
    localPositionзҡ„еҗ«д№үжҳҜеҪ“еүҚRectTransformзҡ„pivotзӣёеҜ№дәҺиҜҘRectTransformзҡ„зҲ¶иҠӮзӮ№rectзҡ„дёӯеҝғзӮ№зҡ„дҪҚзҪ®, з”ұдәҺеҸҜд»Ҙи°ғж•ҙзҲ¶иҠӮзӮ№зҡ„rect, жүҖд»ҘеҸҜд»Ҙй—ҙжҺҘеҪұе“Қеӯҗзү©дҪ“зҡ„localPositionеұһжҖ§, иҖҢжҲ‘д№ҹеҸҜд»Ҙи®©rectзҡ„з»қеҜ№дҪҚзҪ®дёҚеҸҳ йҖҡиҝҮж”№еҸҳpivotзҡ„дҪҚзҪ®жқҘеҪұе“ҚlocalPositionзҡ„еҖј, иҝҷеңЁTransform(йқһRectTransform)дҪ“зі»дёӢжҳҜдёҚеӨӘеҸҜиғҪеҒҡеҲ°зҡ„,

й”ҡзӮ№(Anchor)жҸҸиҝ°зҡ„жҳҜеҪ“еүҚUIиҠӮзӮ№е’Ңе…¶зҲ¶иҠӮзӮ№зҡ„дҪҚзҪ®е…ізі», й”ҡзӮ№иҷҪз„¶еҸ«зӮ№, дҪҶжҳҜдјҡеӯҳеңЁй”ҡзӮ№зҡ„еӣӣдёӘи§’жІЎжңүйҮҚеҗҲзҡ„жғ…еҶө. еҸӘдёҚиҝҮй”ҡзӮ№зҡ„еӣӣи§’йҮҚеҗҲж—¶anchor reference positionе’ҢиҜҘзӮ№йҮҚеҗҲдәҶиҖҢе·І unityдјҡи®Ўз®—еҮәдёҖдёӘanchor reference point, е’ҢpivotеҒҡиҝҗз®—еҫ—еҲ°anchored position
